# Avatars

Drop one JPG per user, named after their first name in lowercase:

- `petrica.jpg`
- `clara.jpg`
- `gabi.jpg`

The `Avi` component picks them up automatically. The face is auto-centered for small/round contexts (TopBar chip, leaderboard rows, profile popover) and the whole image shows on the login picker.

If a PNG is missing, the avatar falls back to a colored initial — no broken icons.
