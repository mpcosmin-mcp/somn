## Architecture (live · refreshed 2026-06-25)

**Tagline**: Gamified social sleep tracker for a 3-person IT team in Sibiu — Sleep Score, REM, RHR, HRV, Duration, with leaderboard + inline social reactions, team charts, and per-user disco-themed avatars. Production: https://somn-xi.vercel.app. Installable as a PWA.

**Stack**: Next.js 16 (App Router, Turbopack) + React 19, Tailwind v4, Geist fonts, `@vercel/kv` for social (reactions + comments). Google Sheets (Apps Script endpoint) for sleep entries. Vercel auto-deploy on `master`. No Anthropic SDK. Minimal service worker for PWA installability.

**Entry points**:
- `/` ([src/app/page.tsx](src/app/page.tsx)) — single centered column (`max-w-3xl`): KPI cards → Leaderboard (with inline social) → Team chart. Click a KPI → `MetricDetailModal`.
- `/api/sheets` — read/write through Apps Script proxy
- `/api/social/{likes,comments}` — KV-backed reactions + threaded comments
- `/manifest.json` + `/sw.js` — PWA install surface

**Key files**:
- [src/lib/sleep.ts](src/lib/sleep.ts) — **source of truth** for `SleepEntry` type, color scales, aggregation, duration helpers
- [src/lib/social.tsx](src/lib/social.tsx) — `SocialProvider` + `useSocial()` hook, `entryKeyOf(date, name)` for reaction/comment lookup
- [src/lib/gamify.ts](src/lib/gamify.ts) — XP, levels, tiers, streaks
- [src/lib/entries-provider.tsx](src/lib/entries-provider.tsx) — context provider; fetches + caches entries (60s server cache + localStorage client cache)
- [src/components/dashboard/leaderboard.tsx](src/components/dashboard/leaderboard.tsx) — period-scoped leaderboard with inline journal text + `EntryReactions` per row, Modal for player detail
- [src/components/dashboard/entry-reactions.tsx](src/components/dashboard/entry-reactions.tsx) — full social UI: `ReactionBar` + comment toggle + threaded comments + composers
- [src/components/dashboard/reaction-bar.tsx](src/components/dashboard/reaction-bar.tsx) — emoji reaction picker (❤️🔥😂😴💪💀)
- [src/components/ui/modal.tsx](src/components/ui/modal.tsx) — centered modal primitive (focus-trap, scroll-lock, responsive)
- [src/components/layout/top-bar.tsx](src/components/layout/top-bar.tsx) — brand + install + library (ReadingList in Modal) + theme + profile
- [src/components/dashboard/player-drawer.tsx](src/components/dashboard/player-drawer.tsx) — player detail content (stats + history + Cheer button)

**Data flow**: `EntriesProvider` fetches from `/api/sheets` → context exposes `{ entries, refetch }`. Social reactions/comments hit `/api/social/*` via Vercel KV. `SocialProvider` wraps the app for optimistic-first social state with localStorage fallback. Leaderboard scopes entries by period (Azi/7z/30z/Total) and shows each player's latest entry's journal + reactions inline.
